## Sizif-512 extension

Extension board for [Sizif-512](https://github.com/UzixLS/zx-sizif-512).

[![photo](images/revA.small.jpg)](images/revA.jpg?raw=true)

### Features
* Turbo Sound FM
* General Sound (12MHz / 512Kb RAM)
* SAA1099
* MIDI sound
